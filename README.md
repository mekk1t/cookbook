# Трапезная

Трапезная - это агрегатор кулинарных рецептов. Наработки базируются на предыдущем подходе KitBook.

Приложение написано на С# с использованием MS SQL для хранения данных и фреймворка Razor Pages для верстки интерфейса. Также для некоторых операций используется и API-функционал.
У проекта есть свой макет на фигме: https://www.figma.com/file/6Vxt6GwDsFQ9wMssMKT68D/Master-Chef?node-id=6%3A314  
Задачи ведутся на https://dev.azure.com/
